# PID-Voltage-Stability-and-Fault-Detection-System
PID-based voltage stability and fault detection system developed using MATLAB/Simulink for stability analysis, disturbance response evaluation, and control system performance assessment.

## Overview

This project presents the design and simulation of a PID-Based Voltage Stability and Fault Detection System using MATLAB/Simulink. The objective is to maintain voltage stability under varying operating conditions and detect abnormal system behavior through voltage deviation analysis.

A closed-loop control architecture was implemented using a PID controller to improve dynamic performance, reduce settling time, minimize overshoot, and enhance system stability.

---

## Objectives

- Design a closed-loop voltage regulation system.
- Implement a PID controller for voltage stabilization.
- Simulate load disturbances and fault conditions.
- Analyze system response characteristics.
- Detect abnormal voltage deviations using fault detection logic.

## Methodology

1. Developed a mathematical model of the system.
2. Designed a PID controller using MATLAB/Simulink.
3. Applied step and disturbance inputs.
4. Evaluated transient and steady-state performance.
5. Implemented threshold-based fault detection.
6. Analyzed controller effectiveness during fault conditions.
